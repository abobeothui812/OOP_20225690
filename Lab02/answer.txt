Question:
- If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
Trả lời : Có, Java cho phép bạn tạo nhiều phương thức khởi tạo trong một lớp, mỗi phương thức có danh sách tham số khác nhau. Khái niệm này được gọi là nạp chồng phương thức khởi tạo.
